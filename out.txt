usage: shuf.py [-h] [-n HEAD_COUNT] [-r] [-e] [-i INPUT_RANGE] [filename ...]

Randomly permute lines of a file

positional arguments:
  filename              input lines or a filename

options:
  -h, --help            show this help message and exit
  -n, --head-count HEAD_COUNT
                        output at most COUNT lines
  -r, --repeat          allow output lines to be repeated
  -e, --echo            treat each ARG as an input line
  -i, --input-range INPUT_RANGE
                        treat each number LO-HI as an input line
